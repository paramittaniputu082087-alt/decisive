# Example Decision Scenario

## Context

A company needs to select a cloud provider for a new platform deployment.

## Options

- AWS
- Azure
- Google Cloud

## Evaluation Criteria

- Cost
- Reliability
- Security
- Compliance

## Evaluation Result

Azure receives the highest reliability and compliance score.

## Decision Outcome

Decision Approved: Azure selected.

## Governance Record

Decision recorded in the decision ledger with full evaluation trace.
