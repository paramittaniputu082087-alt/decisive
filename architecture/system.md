# Decisive System Architecture

Decisive acts as a decision intelligence layer between decision sources and execution systems.

Main Components

Decision Input Layer  
Receives decision requests from humans, AI systems, or workflows.

Decision Model Engine  
Structures the decision context and available options.

Evaluation & Scoring Layer  
Evaluates options using defined criteria.

Policy & Governance Engine  
Applies organisational policies and risk rules.

Decision Outcome Layer  
Determines whether the decision is approved, escalated, or blocked.

Decision Ledger  
Stores audit records and decision outcomes.
